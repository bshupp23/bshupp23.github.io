HTML & CSS Cheatsheet

Table Tags

Tag Name Description
<table> Table The wrapper element for all HTML tables.
<thead> Table Head The set of rows defining the column headers in a table.
<tbody> Table Body The set of rows containing actual table data.
<tr> Table Row The table row container.
<td> Table Data The table row container.
<tfoot> Table Foot The set of rows defining the footer in a table.
<caption> Caption Defines a caption for the table, appearing above the table.

Table Attributes

Attribute Name Description
colspan Column Span Defines how many columns a td element should span.
rowspan Row Span Defines how many rows a td element should span.

Basic Structure

Tag Name Description
<!DOCTYPE html> DOCTYPE html Defines the document type as HTML5.
<html> HTML Root element of the HTML document.
<head> Head Contains meta information about the document (title, stylesheets, scripts).
<title> Title Defines the title displayed in the browser tab.
<body> Body Contains the visible content of the webpage.

Headings

Tag Name Description
<h1> h1 Heading of first importance
<h2> h2 Heading of second importance
<h3> h3 Heading of third importance
<h4> h4 Heading of fourth importance
<h5> h5 Heading of fifth importance
<h6> h6 Heading of sixth importance

Content Elements

Tag Name Description
<p> Paragraph Defines a paragraph of text.
<br> Line Break Inserts a line break within a paragraph.
<span> Span Defines an inline element for applying styles without creating a new section.
<a> Anchor Creates a hyperlink to another webpage or section within the same page.

Lists

Tag Name Description
<ul> Unordered List Defines an unordered list (bullet points).
<ol> Ordered List Defines an ordered list (numbered).
<li> List Item Defines a list item within ol or ul.

Images & Img Attributes

Tag Name Description
<img> Image Defines an image element.
src Source Source of the image
alt Alt Alternative text for accessibility

Forms

Tag Name Description
<form> Form Defines a form for collecting user input.
<input> Input Defines an input field (text, password, checkbox, etc.).
<label> Label Associates a label with a form element.
<button> Button Defines a clickable button that can submit the form or trigger an action.
<textarea> Text Area Defines a multi-line text input field.
<select> Select Defines a drop-down selection list with options.
<option> Option Defines an option within a <select> element.
<fieldset> Fieldset Groups related form elements visually.
<legend> Legend Defines a caption for a <fieldset>

Semantics & Content

Tag Name Description
<article> Article Defines an independent, self-contained piece of content (e.g., blog post, news article).
<section> Section Defines a generic section of a webpage with a common theme.
<aside> Aside Defines content that's indirectly related to the main content (e.g., sidebar).
<header> Header Defines the header section of a webpage, often containing logo, navigation, etc.
<footer> Footer Defines the footer section of a webpage, often containing copyright, contact information, etc.
<b> Bold Bold
<i> Italic Italic
<strong> Strong Bold for Importance
<em> Italicize Italicize
<sup> Superscript Used for superscript text
<sub> Subscript Used for subscript text
<pre> --- Defines preformatted text, preserving whitespace and line breaks.

Embedding Content

Tag Name Description
<iframe> iframe Embeds an external resource (e.g., video from YouTube) within the webpage.
<audio> Audio Defines an audio element (include source file info and controls).
<video> Video Defines a Video element (include source file info and controls).

Other

Tag Name Description
<meta> Meta Defines meta information about the document, not displayed on the page itself.
<link> Link Links the document to external resources like stylesheets or favicons.
<style> Style Defines inline styles for the document within the <head> section.
<script> Script Defines scripts using JavaScript or other languages, placed in the head or body.

CSS Cheatsheet

Text Properties

Name Description
font-family Specifies the font family for the element's text.
font-size Sets the size of the text.
font-weight Controls the boldness of the text (normal, bold, bolder, etc.).
font-style Defines the style of the text (normal, italic, oblique).
text-decoration Controls underline, overline, strikethrough, etc.
text-align Aligns the text within its container (left, right, center, justify).
line-height Sets the height of a line of text.

Colors & Backgrounds

Name Description
color Defines the text color or background color of an element.
background-color Sets the background color of an element.
background-image Sets an image as the background of an element.
background-repeat Controls how a background image repeats (repeat, no-repeat, etc.).
